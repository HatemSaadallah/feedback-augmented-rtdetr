RT-DETR achieves real-time object detection without non-maximum suppression by combining a hybrid encoder with a transformer decoder, but its small-object performance ($\mathrm{AP}_S$) lags both its larger-object scores and several CNN-based detectors at comparable speeds. We propose a \emph{decoder-to-encoder feedback} module that refines encoder memory mid-decoding using the predictions of the first decoder layer, providing later decoder layers with a representation already conditioned on the rough object hypotheses. A first implementation (v1) added the module with a sigmoid-parameterized scalar gate applied to all four feature levels and produced the expected on-paper improvement, but a same-checkpoint inference-time ablation revealed a delta of $0.0$ on $\mathrm{AP}_S$: the gate had decayed to ${\approx}0.12$ during training, silencing the mechanism. A revised implementation (v2) addresses this with two structural changes: a \emph{gate floor} reparameterization that prevents the optimizer from collapsing the gate below a fixed minimum, and a \emph{level mask} that restricts cross-attention to the P2 and P3 features where small objects live. v2 yields a measurable causal contribution of $\Delta\mathrm{AP}_S = +0.99$ ($33.26 \to 34.25$) at $640\!\times\!640$ inference, with no added parameters relative to v1. We report full ablation tables, per-epoch trajectories, gate dynamics, and qualitative detections on COCO val2017.
